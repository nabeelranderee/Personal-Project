# Personal Project - Date Plans Website

A beautifully designed, interactive date planning website built with vanilla HTML, CSS, and JavaScript.

## Features

✨ **Interactive Name Gate** - Personalized entry experience
💕 **Heart Selection System** - Like your favorite date ideas
🎨 **Smooth Animations** - Confetti effects and transitions
📱 **Responsive Design** - Works on desktop and mobile
🎯 **Vibe Check Quiz** - Personalized recommendations
🌶️ **Romantic Aesthetic** - Curated color scheme and typography

## How It Works

1. Enter your name at the entry gate
2. Take the vibe check quiz to get personalized recommendations
3. Heart your favorite date ideas (no limit!)
4. Click "Send my picks" to see your selections
5. Screenshot and share your choices

## Dates Included

10 curated date ideas in Johannesburg, including:

- And more...

## Technology Stack

- **HTML5** - Semantic structure
- **CSS3** - Variables, animations, gradients, grid layout
- **Vanilla JavaScript** - Interactive features (no frameworks)
- **Google Fonts** - Lora, Plus Jakarta Sans, Caveat

## Customization

Edit `index.html` to:
- Change names from "Nabeel & You" to your own
- Update the chilli lane reference
- Modify date ideas and locations
- Adjust colors via CSS variables in `:root`
